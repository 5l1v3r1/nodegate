# Modifiers

All modifiers can be accessed from the top level function of **nodegate** or by direclty importing
the modifiers path:

```js
const nodegate = require('nodegate');
const { aggregate } = nodegate;
// or
const { aggregate } = require('nodegate/modifiers');
```
